# Roadmap

## เสร็จแล้ว
- **v0.1.1** — Swap โพรโทไทป์ (ไฟล์เดียว)
- **v0.2.0 Alpha** — Refactor เป็นสถาปัตยกรรมเอนจิน: ES6 Classes, engine/board/systems, mobile-first canvas + อาร์ต 8-bit/CRT ✅
- **v0.2.1** — Match detection + Gravity + Cascade + สลับกลับเมื่อไม่ match + ชุดเทสต์ ✅
- **v0.2.2** — ScoreSystem (chips × mult) + HUD + กันเกมตัน + ปรับสีส้ม/ขนาดสไปรต์ ✅
- **v0.2.3** — ลูกกวาดพิเศษ: 💣 ระเบิด 3x3 ลูกโซ่, 🌟 ซูเปอร์โนวา + โบนัสคะแนน ✅
- **v0.2.4** — เอฟเฟกต์: พาร์ติเคิล, จอสั่น, ตัวเลขคะแนนลอย + เสียงสังเคราะห์ WebAudio ✅
- **v0.3.0** — ① ปัดสลับ (Swipe Input, threshold 28px, รองรับ 4 ทิศ) + แปล HUD/UI เป็นไทยครบ ✅
- **v0.3.1** — ④ HUD Redesign (กลุ่มพาเนลมนโปร่งแสง + ไอคอน + ลดความสูง) + ⑤ เพิ่มระยะห่างกระดาน ✅
- **v0.4.0** — ② ขัดลูกกวาด (หายใจ + เรืองแสง) + ③ ฟีดแบ็กเลือกช่องแบบพัลส์ + ⑥ พื้นหลัง (พารัลแลกซ์/ฝุ่น/ดาวตก/ดาวเคราะห์ไกล) + ⑦ อนิเมชัน (ease-in-out, squash&stretch, landing bounce) + ⑧ Game Feel รวม ✅
- **v0.4.1** — ฟอนต์ Chakra Petch สำหรับข้อความไทยทั้งหมด (Press Start 2P ไม่มีตัวอักษรไทย ทำให้ข้อความไทยเคยเพี้ยน/ไม่เข้าธีม) ✅ (ตอนนี้)

> ครบทุกข้อใน **TOP PRIORITIES** ของ Master Development Document แล้ว (①-⑧) — โฟกัสต่อไปนี้คือ Milestone 3 ตาม Roadmap เดิม
