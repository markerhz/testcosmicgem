### (ก) จรวดแยกทิศ + รื้อ mapping ตัวพิเศษ
- **เรียง 4 แนวนอน → 🚀 จรวดแถว (rocketH)** ล้างทั้งแถว | **เรียง 4 แนวตั้ง → 🚀 จรวดคอลัมน์ (rocketV)** ล้างทั้งคอลัมน์
- **รูป L/T (จุดตัดแนวนอน×แนวตั้ง) → 💣 ระเบิด 3x3** (ย้ายจากเรียง 4 เดิม)
- **เรียง 5 → 🌟 โนวา** ล้างทั้งสี (คงเดิม)
- อาร์ต: สไปรต์บีมทิศเดียว 2 แบบ (นอน/ตั้ง) + หัวลูกศรบอกทิศ

### (ข) คอมโบสลับ 2 ตัวพิเศษ (activateSpecialSwap)
- จรวด+จรวด = ล้างแถว+คอลัมน์ (กากบาทเต็ม) | จรวด+ระเบิด = 3 แถว+3 คอลัมน์ | ระเบิด+ระเบิด = 5x5

- **สลับตัวพิเศษเดี่ยว+เม็ดปกติ = จุดชนวนในที่** (จรวดยิงแนว, ระเบิดบึ้ม 3x3) — ไม่ต้องรอแมตช์
- คอมโบจุดชนวนตัวพิเศษอื่นในพื้นที่ต่อเป็นลูกโซ่ผ่าน expand loop เดิม

### เทส & สโคป

## 🔊 Sprint 3 / TASK-015: เสียง + Game Feel รอบเก็บ — 2026-07-11

### ยกเครื่องเสียงให้ตรง AUDIO_GUIDE (อุ่น/กลไก/นุ่ม — never arcade)
- **สถาปัตยกรรมเสียงใหม่** — ทุกเสียงผ่าน lowpass ตัดความบาดหู + มี attack สั้นกันเสียงคลิก + ใช้ sine/triangle เป็นหลัก + master bus กันเสียงซ้อนแตก
- **คำศัพท์เสียงกลไก/คริสตัล**: `select`=tick เบา, `swap`=light clunk ทุ้ม, `invalid`=ปฏิเสธนุ่มลงต่ำ (เลิก buzzer), `pop`=คริสตัลแตก impact อุ่น+shimmer สูง (พิตช์ไต่คอมโบแบบคุมไว้), `bomb`=heavy impact ทุ้มหนัก+รัมเบิลกรองต่ำ (เลิก sawtooth แหลม), `nova`=คอร์ด C major บานอุ่น+ประกายค้าง (relaxing bloom)
- เดิมเสียงเป็น square/sawtooth บี๊บอาร์เคด → ตอนนี้อุ่นเข้าธีมยานอวกาศ cozy

### Game Feel
- **เสียงลงจอด (`land`)** — เม็ดตก/เครื่องขุดปล่อยเม็ดมี light clunk กลไกเบาๆ ครั้งเดียวต่อคลื่น (ไม่สแปม) เพิ่มน้ำหนักการตก
- แนบ `sfx_audition.html` (standalone) ไว้ฟังทุกเสียงก่อนตัดสินใจ

### หมายเหตุ
- Sfx ไม่อยู่ในชุดเทส (WebAudio ต้องมีเบราว์เซอร์) — คง API เดิมครบ Game.js เรียกเหมือนเดิม + เพิ่ม `land()`
