# Changelog

## v0.2.4 Alpha — 2026-07-10

### เพิ่ม
- **Effects.js** — ระบบพาร์ติเคิล (ระเบิดสีลูกกวาดตอนแตก, ตัวพิเศษระเบิดหนักกว่าปกติ 2 เท่า), เลขคะแนนลอยขึ้น ("+gained" สีทองถ้าคอมโบสูง), จอสั่น (แรงขึ้นตามระเบิด/โนวา/จำนวนช่องที่แตก) — เป็นข้อมูลล้วนเหมือน Board/Animation ไม่แตะ canvas เอง
- **Sfx.js** — เสียงสังเคราะห์ด้วย WebAudio ล้วน ไม่มีไฟล์เสียง: แตะเลือก, สลับ, สลับกลับ (ผิด), pop (เสียงสูงขึ้นตามคอมโบ), ระเบิด (โทนต่ำ + white noise), โนวา (คอร์ดไล่เสียง)
- **ปุ่มปิด/เปิดเสียง** ใน HUD (🔊/🔇), AudioContext resume หลัง gesture แรกของผู้ใช้ตามข้อจำกัดเบราว์เซอร์
- `Renderer.draw()` รับ `effects` เพิ่ม: ใช้จอสั่นขยับ transform ก่อนวาด, วาดพาร์ติเคิลพิกเซล + เลขลอยก่อนชั้น CRT (CRT overlay ไม่ขยับตามจอสั่น ให้ฟีลกล้องสั่นในตู้จริง)
- เทสต์เพิ่มเป็น 43 ข้อ (Effects: burst/หมดอายุ, floatText, จอสั่น + การทับกันของความแรง)

### ข้อจำกัดที่รู้อยู่แล้ว
- ยังไม่มีด่าน + Joker (v0.3.0)

## v0.2.3 Alpha — 2026-07-09

### เพิ่ม
- **💣 ระเบิด** — เรียง 4 ตัว สร้างระเบิดตรงช่องที่สลับ, แตกเมื่อไหร่กวาด 3x3 รอบตัว, ระเบิดโดนกันเอง = ลูกโซ่
- **🌟 ซูเปอร์โนวา** — เรียง 5 ตัว, สลับกับสีไหนล้างสีนั้นทั้งกระดาน, โนวา+โนวา = ล้างทั้งกระดาน, โดนลูกหลงจะล้างสีสุ่ม 1 สี
- **สไปรต์ตัวพิเศษแบบพิกเซล** — ระเบิด: แกนไฟเต้นตุบๆ 2 เฟรม / โนวา: ดาว 4 แฉกสีหมุนวน 3 เฟรม (สร้างด้วยโค้ดล้วนเหมือนเดิม)
- **โบนัสคะแนน** — ระเบิดทำงาน +50 chips, โนวาทำงาน +100 chips ต่อดวง
- `MatchSystem.planClears()` + `expandClears()` (พอร์ตจากลอจิกที่ผ่านเทสต์) — โนวาไม่จับคู่แบบปกติ
- เทสต์เพิ่มเป็น 33 ข้อ

### ข้อจำกัดที่รู้อยู่แล้ว
- ยังไม่มีเอฟเฟกต์พาร์ติเคิล/จอสั่น/เสียง (v0.2.4), ด่าน + Joker (v0.3.0)

## v0.2.2 Alpha — 2026-07-09

### เพิ่ม
- **ScoreSystem** — คะแนน = chips × mult (เม็ดละ 10 แต้ม, cascade เพิ่มตัวคูณชั้นละ +0.5) พร้อม HUD สไตล์ 8-bit แสดง SCORE + MULT (ตัวคูณเปลี่ยนสีตอนคอมโบ)
- **กันเกมตัน** — `MatchSystem.hasPossibleMove()` ลองสลับทุกคู่แบบไม่แตะกระดานจริง, ตันเมื่อไหร่สับกระดานใหม่อัตโนมัติ (`Board.fillRandom()`)
- เทสต์เพิ่มเป็น 23 ข้อ (ScoreSystem, กระดานตันจริง, reshuffle)

- **ลูกกวาดใหญ่ขึ้น** — รัศมีสไปรต์ 6.5 → 7.4 พิกเซล (เต็มช่องขึ้น ลายในปรับตามสัดส่วน)

### ข้อจำกัดที่รู้อยู่แล้ว

## v0.2.1 Alpha — 2026-07-09

### เพิ่ม
- **MatchSystem.findMatches()** — ตรวจจับ run แนวนอน/แนวตั้ง >= 3 + `collectCells()` รวมช่องแบบไม่ซ้ำ (รองรับรูป L/T)
- **GravitySystem** — `applyGravity()` ลูกกวาดหล่นเติมช่องว่าง, `refill()` เกิดใหม่จากด้านบน
- **Cascade loop ใน Game** — แตก → หล่น → เติม → เช็คใหม่ วนจนนิ่ง พร้อมอนิเมชันครบ (หด, หล่น, spawn จากเหนือจอ)
- **สลับกลับอัตโนมัติ** เมื่อสลับแล้วไม่เกิด match
- **กระดานตั้งต้นไม่มี match** — สุ่มแบบเลี่ยงเรียง 3 ตั้งแต่แรก
- **ชุดเทสต์** `tests/test.js` (14 เทสต์, seeded rng ให้ผลซ้ำได้) + `package.json` รัน `npm test` ได้

### ข้อจำกัดที่รู้อยู่แล้ว
- ยังไม่มีคะแนน (v0.2.2), ลูกกวาดพิเศษ (v0.2.3), เอฟเฟกต์/เสียง (v0.2.4)
- ยังไม่กันเกมตัน (hasPossibleMove — v0.2.2)

## v0.2.0 Alpha — 2026-07-09

### เพิ่ม
- **Refactor ทั้งโปรเจกต์เป็นสถาปัตยกรรม ES6 Classes** แยกความรับผิดชอบชัดเจน ไม่มีตัวแปร global
- `engine/Game.js` — state machine (IDLE / ANIMATING), ลูปหลัก rAF, เชื่อมทุกระบบ
- `engine/Renderer.js` — วาด 60 FPS, mobile resize + devicePixelRatio, พื้นหลังอวกาศ prerender, ลูกกวาดเรืองแสง 6 แบบ
- `engine/Input.js` — pointer event → พิกัดช่อง (รองรับเมาส์และทัชในตัว)
- `engine/Animation.js` — tween manager แบบ Promise-based พร้อม easing
- `board/Board.js`, `board/Cell.js`, `board/Candy.js` — โครงสร้างข้อมูลกระดาน 8x8, 6 ชนิดลูกกวาด
- Selection: แตะเลือก + ไฮไลต์กรอบเต้น, แตะซ้ำยกเลิก, แตะไกลย้ายการเลือก
- Swap: สลับช่องติดกันพร้อมอนิเมชันนุ่ม, ล็อกอินพุตระหว่างอนิเมชัน
- System stubs พร้อม interface และแผนเวอร์ชัน: `MatchSystem`, `GravitySystem`, `ScoreSystem`, `SaveSystem`
- เอกสาร: README, ROADMAP, GDD, CHANGELOG
- **อาร์ตสไตล์ 8-bit + CRT (แบบ Balatro)**: สไปรต์พิกเซล 16x16 สร้างด้วยโค้ดล้วน (`Renderer.spriteData` เป็น pure function เทสต์ได้), ขยาย 4 เท่าไม่เกลี่ยพิกเซล, เส้นสแกน + ขอบจอมืดแบบ CRT, ดาวกะพริบแบบขั้นบันได, เนบิวลา dither, ฟอนต์พิกเซล Press Start 2P, กรอบเลือกมุมกะพริบสไตล์เรโทร

### ข้อจำกัดที่รู้อยู่แล้ว (ตั้งใจตามแผน Alpha)
- ยังไม่มี match detection — สลับแล้วค้างตำแหน่งใหม่ ไม่มีการตรวจ/สลับกลับ
- ยังไม่มี gravity, cascade, คะแนน, ลูกกวาดพิเศษ, เอฟเฟกต์, เสียง, เซฟเกม
- ต้องรันผ่านเว็บเซิร์ฟเวอร์ (ES Modules เปิดแบบ file:// ไม่ได้)
