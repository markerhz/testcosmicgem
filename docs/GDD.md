# Game Design Document — GemVerse

## แนวคิด
Match-3 **กึ่งแอคชั่น** ตีมลูกกวาดอวกาศ จุดขายคือ **ตัวเปลี่ยนเกม (Joker)**
ที่ทำให้วิธีคิดคะแนนเปลี่ยนทุกรอบ — แรงบันดาลใจจากไพ่ Joker ในเกม Balatro
ผสมความมันของ match-3 สายเอฟเฟกต์แบบ Candy Crush / Shopee Candy

## แกนเกมเพลย์ (Core Loop)
1. สลับลูกกวาด → match → cascade → ได้คะแนน
2. ทำคะแนนถึงเป้าก่อนหมดตาเดิน → ผ่านด่าน
3. ผ่านด่าน → **เลือกการ์ด Joker 1 จาก 3 ใบ** → เป้าด่านถัดไปโหดขึ้น
4. วนไปเรื่อยๆ จนตาเดินหมด = จบเกม (นับสถิติว่าไปได้ไกลแค่ไหน)

## ระบบคะแนน (แบบ Balatro)
```
คะแนนต่อสเต็ป = chips × mult
```
- **chips** = แต้มฐานรวมของลูกกวาดที่แตก (เม็ดละ 10 + โบนัส Joker)
- **mult** = 1 + (cascade chain − 1) × 0.5 + โบนัส Joker
- เป้าด่าน: `500 × 1.8^(ด่าน−1)` — โตแบบทวีคูณ บังคับให้ต้องพึ่ง Joker

## ลูกกวาด (6 ชนิด ตีมอวกาศ)
| ชนิด | รูป | สี |
|---|---|---|
| 0 | ดาวเคราะห์มีวงแหวน | แดง |
| 1 | ดาวห้าแฉก | เหลือง |
| 2 | คริสตัล | เขียว |
| 3 | จันทร์เสี้ยว | ฟ้า |
| 4 | วงโคจร | ม่วง |
| 5 | ดาวหาง | ส้ม |

## ลูกกวาดพิเศษ
- **เรียง 4 → 💣 ระเบิด**: แตกแล้วกวาด 3x3 รอบตัว, ระเบิดชนกัน = ลูกโซ่
- **เรียง 5 → 🌟 ซูเปอร์โนวา**: สลับกับสีไหน เคลียร์สีนั้นทั้งกระดาน

## Joker (ตัวอย่าง 9 ใบแรก — ถือได้สูงสุด 5)
| การ์ด | ผล |
|---|---|
| 🔴 ดาวยักษ์แดง | ลูกกวาดแดง +15 chips |
| 🌙 นักขุดดวงจันทร์ | ลูกกวาดฟ้า +15 chips |
| 🔭 นักดูดาว | ทุกสี +5 chips |
| 🕳️ หลุมดำ | สเต็ปที่มีระเบิด mult +1.0 |
| 🌀 คอมโบจักรวาล | cascade ชั้นละ +0.5 mult เพิ่ม |
| 💜 หัวใจเนบิวลา | mult พื้นฐาน +0.5 |
| ☄️ ดาวหางนำโชค | 20% คะแนนสเต็ป x2 |
| ⏳ บิดเวลา | เริ่มด่าน +3 ตาเดิน |
| 🌟 แฟนซูเปอร์โนวา | สร้างระเบิด/โนวา +100 คะแนนทันที |

## ฟีล "กึ่งแอคชั่น"
- ลากปัดเพื่อสลับเร็วๆ ได้ (นอกจากแตะ 2 ครั้ง)
- เอฟเฟกต์จัดเต็ม: พาร์ติเคิล, จอสั่นตอนระเบิด, ตัวเลขลอย, ข้อความ COMBO
- เสียงสังเคราะห์ WebAudio ระดับเสียงสูงขึ้นตาม chain (ไม่ต้องมีไฟล์เสียง)

## แพลตฟอร์ม
- ตอนนี้: HTML5 mobile-first (เล่นในเบราว์เซอร์)
- อนาคต: แอพ Android ผ่าน Capacitor/TWA, ระบบออนไลน์ (leaderboard, daily challenge)
